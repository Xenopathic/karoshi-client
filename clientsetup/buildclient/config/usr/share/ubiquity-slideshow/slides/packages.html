Pre-loaded with useful applications

Karoshi-Client is full of applications pre-installed for your convenience. You can access them using the improved menu at the top left corner of the desktop. Here is a list of some of the applications you can find.

Development

Atom, a hackable text editor for the 21st Century.

Git, open-source version control system.

Eclipse, Java IDE.

Web Browsing

Firefox, web browser by Mozilla.

Thunderbird, mail client by Mozilla.

Networking Tools

Filezilla, open-source FTP client.

Wireshark, network protocol analyzer.

Office

Libreoffice, open-source office suite.

Graphics

Gimp, open-source graphics editor.

Inkscape, open-source professional vector graphics editor.

Misc

VLC, universal media player.

Stellarium, open-source planetarium for your computer.
